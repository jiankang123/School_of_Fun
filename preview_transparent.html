🦞 OpenClaw 是什么？

OpenClaw（原名 Moltbot、Clawdbot）是一个本地运行的 AI 个人助理，可以通过微信、飞书、Telegram、WhatsApp 等聊天工具控制。

简单理解：给你一种做皇帝的感觉，动动嘴就能让 AI 帮你干活，情绪价值拉满。

📧收发邮件、管理日历
🌐浏览网页、搜索截图
📝写代码、改文档、管项目
🔧通过 Skills 扩展无限可能
💡 核心理念：好的 OpenClaw = 调教 + 优化 + 记忆系统。稳定 > 功能多！
✅ 方法一：WSL2 安装（Windows 推荐）

官方强烈建议在 Windows 上使用 WSL2，直接安装可能有兼容性问题。

  • 1以管理员身份打开 PowerShell，启用 WSL2：
    • wsl --install

    重启电脑后，WSL2 会自动安装 Ubuntu。

  • 2打开 Ubuntu 终端，安装 OpenClaw：
    • curl -fsSL https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
openclaw gateway status
openclaw dashboard
    ⚡ 方法二：Windows 直接安装
    ⚠️ 需要 Node.js 22.x 或更高版本。访问 nodejs.org 下载 LTS 版本安装。

    打开 PowerShell（不是 CMD），运行：

    iwr -useb https://openclaw.ai/install.ps1 | iex

    然后运行安装向导：

    openclaw onboard --install-daemon
    📦 方法三：npm 手动安装

    如果安装脚本出现问题，可以手动用 npm 安装：

    npm install -g openclaw@latest
openclaw onboard --install-daemon
    📋 常用命令速查表

    ⚠️ 新手必看：常见坑
    🖥️ 不要用 MacBook 当 24 小时服务器，建议部署到云服务器（阿里云、腾讯云等）。
    😴 防止 Mac 休眠断联：只要终端开着就不会进入休眠：
    caffeinate -i
    🛡️ 注意安全：官方插件网站有很多病毒 Skill，安装时要注意来源！
    🔑 openclaw 命令找不到？执行 npm prefix -g 获取路径，将该路径添加到系统 PATH 环境变量。
    💻 WSL2 安装后无法启动？确保 BIOS 中已启用虚拟化，且 Windows 版本为 1903 及以上。
    🔧 首次配置：接入飞书
  • 1在飞书开放平台创建企业自建应用
  • 2获取 App ID 和 App Secret
  • 3配置事件订阅和回调地址（长链接模式）
  • 4在 OpenClaw 中配置飞书通道，执行配对
  • 5发布应用，给机器人发消息测试
    • 📹 推荐先看视频教程：「30 分钟把 OpenClaw 在飞书上配到可用」
    🧩 什么是 Skills？

    Skills（技能）就像给龙虾加上了利索的手和眼，必不可少！通过安装不同的 Skill，龙虾可以收发邮件、操控浏览器、管理文件等。

    💡 好用的 Skill = 给龙虾配置手、脚、耳朵。优先搞 Skills，再考虑多 Agent。
    📦 安装 Skills
    # 从 ClawHub 安装技能
openclaw skills install [技能名]

# 查看已安装技能
openclaw skills list

# 禁用某个技能
claw disable [技能名]
    ⚠️ 遇到 Rate limit exceeded？暂停 5-10 分钟后重试，或直接从 ClawHub 下载 ZIP 包手动解压到 ~/.openclaw/skills。
