## 🚀 核心项目

### 1. 数学启蒙星球 (math-mp)

专为 3-12 岁儿童设计的数学启蒙微信小程序。

**核心特点：**
- 🎮 游戏化学习，不是刷题
- 📊 分阶段课程（学前班 → 小学）
- ⭐ 星星奖励系统
- 👨‍👩‍👧 家长控制台

**课程阶段：**
| 阶段 | 年龄 | 内容 |
|------|------|------|
| 学前班 | 3-5岁 | 数字认知、形状识别 |
| 幼儿园 | 5-6岁 | 10以内加减法、数字排序 |
| 小学低年级 | 6-8岁 | 100以内计算、乘法口诀 |
| 小学中高年级 | 8-12岁 | 分数、几何、方程入门 |

### 2. 微信小程序 (miniprogram)

完整的微信小程序项目框架。

### 3. 部署站点 (deploy-site)

Web 部署相关资源和配置。

## 🚀 快速开始

### 小程序开发

```bash
# 1. 克隆项目
git clone https://github.com/jiankang123/School_of_Fun.git

# 2. 使用微信开发者工具打开 miniprogram 或 math-mp 目录

# 3. 配置 AppID（在 project.config.json 中）

# 4. 编译运行
```

### Web 站点开发

```bash
# 进入部署站点目录
cd deploy-site

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
