# 🚀 快速导入指南（3 分钟上手）

## 第一步：解压文件

1. 解压 `math-planet-miniprogram-完整版.zip`
2. 得到 `miniprogram/` 文件夹

---

## 第二步：导入微信开发者工具

1. **打开微信开发者工具**（没有的话[点此下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)）

2. **点击「导入」或「+」**

3. **填写项目信息**：
   ```
   项目名称：数学启蒙星球（随意填）
   目录：选择 miniprogram 文件夹
   AppID：填你的 AppID（测试可选「测试号」）
   ```

4. **点击「导入」**

---

## 第三步：配置域名（重要！）

### 📱 真机预览必须配置，模拟器可跳过

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入你的小程序后台
3. **开发 → 开发管理 → 开发设置 → 服务器域名**
4. 点击 **downloadFile** 右侧的「修改」
5. 添加以下两个域名（一行一个）：
   ```
   https://images.unsplash.com
   https://www.pacdv.com
   ```
6. 点击「保存并提交」

### 为什么需要配置？
- `images.unsplash.com` - 加载真实动物图片
- `www.pacdv.com` - 播放动物叫声

---

## 第四步：编译预览

### 在模拟器中预览（立即可用）
1. 开发者工具左上角点击「编译」
2. 在右侧模拟器中体验

### 在真机上预览（推荐）
1. 点击工具栏「预览」
2. 用微信扫描二维码
3. 在手机上打开小程序

---

## ❓ 遇到问题？

### 1. 「不在以下 request 合法域名列表中」
**原因**：域名白名单未配置  
**解决**：回到第三步配置域名

### 2. 「AppID 无效」
**原因**：AppID 填写错误  
**解决**：检查是否复制完整，或使用「测试号」

### 3. 图片不显示
**原因**：网络问题或域名未配置  
**解决**：
- 确认域名已配置（第三步）
- 关闭开发者工具的「不校验合法域名」（调试时可开启）

### 4. 没有声音
**原因**：音频域名未配置或真机静音  
**解决**：
- 确认 `www.pacdv.com` 已添加到 downloadFile 域名
- 检查手机音量
