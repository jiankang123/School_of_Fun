# 数学启蒙星球 - 微信小程序完整项目

## 📦 项目简介

这是一个专为 2-7 岁儿童设计的数学启蒙教育小程序，包含：
- 🌱 **启蒙班**（2-3岁）：认动物、认水果、认颜色等基础认知
- 🍎 **学前班**（3-5岁）：数数、配对等数字概念训练
- 🚂 **幼儿园**（5-6岁）：基础加减法
- 🏫 **小学一年级**（6-7岁）：20以内加减法、填空、比大小

**总计 4 个年龄段，32 个关卡**

---

## 🚀 快速开始

### 1. 导入项目

1. 下载并解压 `miniprogram.zip`
2. 打开**微信开发者工具**
3. 点击「导入项目」→ 选择 `miniprogram` 文件夹
4. 填入你的 **AppID**（测试可用测试号）
5. 点击「确定」

### 2. 配置域名白名单（重要）

登录 [微信公众平台](https://mp.weixin.qq.com/)：

**开发 → 开发管理 → 服务器域名 → downloadFile 合法域名**

添加以下域名（用于加载图片和音效）：
```
images.unsplash.com
www.pacdv.com
```

### 3. 编译预览

- 点击「编译」按钮
- 选择「预览」扫码在真机上体验
- 或直接在模拟器中测试

## 🎮 功能特性

### 1. 启蒙班（2-3岁）
- **认知类游戏**：动物、水果、颜色、形状、交通工具
- **真实图片 + 叫声**：使用 Unsplash 高质量照片和动物叫声
- **题目不重复**：同一关内不会重复出题

### 2. 学前班（3-5岁）
- **数数游戏**：点击计数 + 选择答案
- **连线配对**：数字与点点配对（Canvas 画线）

### 3. 幼儿园（5-6岁）
- **加减法**：10 以内加减法，图示辅助理解

### 4. 小学一年级（6-7岁）
- **进阶计算**：20 以内加减法
- **填空题**：数字键盘输入
- **比大小**：>、=、< 符号选择

## 🔧 技术栈

- **框架**：微信小程序原生开发
- **状态管理**：wx.storage 本地持久化
- **音效**：Web Audio API → WAV 临时文件（真机兼容）
- **图片**：Unsplash API（免费高质量）
- **数据流**：单向数据流（setData）

## 📝 核心文件说明

### `utils/data.js`
关卡数据配置，定义所有游戏内容：
```javascript
{
  name: '认动物①',
  type: 'recognize',
  items: [
    '图片URL#名字#叫声URL',
    'https://images.unsplash.com/photo-xxx#狗#sound.mp3'
  ]
}
```

### `utils/store.js`
全局状态管理：
- `stars` - 累计星星数
- `streak` - 连续天数
- `prog` - 各关卡进度 `{sk-li: {stars, done}}`

### `utils/sfx.js`
音效引擎：
- 纯代码合成 WAV 音频
- 写入临时文件后播放（真机兼容）
- 6 种音效：答对、答错、点击、通关、完美、金币

## 🎨 自定义配置

### 修改主题色

编辑 `app.wxss`：
```css
/* 主色调 */
--org: #FFB300;  /* 橙色 */
--blu: #4FC3F7;  /* 蓝色 */
--grn: #66BB6A;  /* 绿色 */
```

### 添加新关卡

编辑 `utils/data.js`，在对应阶段的 `levels` 数组里添加：
```javascript
{ 
  name: '新关卡', 
  emoji: '🎯', 
  color: '#FF5722', 
  type: 'count',  // 游戏类型
  max: 5 
}
```

### 修改 tabBar 图标

替换 `assets/` 目录下的 PNG 图标（81×81px）

## 🐛 常见问题

### 1. 分享按钮是灰色的？
**原因**：小程序未实现 `onShareAppMessage`  
**解决**：已在所有页面添加分享功能，重新编译即可

### 2. 真机没有声音？
**原因**：真机不支持 data URI 音频  
**解决**：已修复，使用临时文件方式播放

### 3. 连线游戏线条错位？
**原因**：Canvas 坐标计算问题  
**解决**：已改用 CSS 绝对定位 + transform 实现

### 4. 认知游戏所有答案都错？
**原因**：答案比较逻辑错误（字符串被转成数字）  
**解决**：已修复，根据游戏类型智能判断

### 5. 图片加载失败？
**原因**：未配置域名白名单  
**解决**：在小程序后台添加 `images.unsplash.com` 到 downloadFile 域名
