# 分享功能说明

## 已添加的分享功能

小程序所有主要页面都已实现**分享到好友**和**分享到朋友圈**功能：

### 各页面分享内容

| 页面 | 分享标题示例 | 分享路径 |
|------|-------------|---------|
| **主页** | 我在数学启蒙星球已经获得了 20 颗星！一起来挑战吧 ⭐ | `/pages/home/home` |
| **关卡地图** | 🍎 学前班 - 一起来挑战数学启蒙！ | `/pages/home/home` (tabBar 限制) |
| **游戏中** | 🍎 数苹果① - 来挑战这一关！ | `/pages/game/game?sk=pre&li=0` |
| **通关奖励** | 🏆 获得了 30 颗星！一起来挑战数学启蒙星球吧 | `/pages/home/home` |
| **进度页** | 📊 看看我的数学学习进度！一起来玩数学启蒙星球 | `/pages/home/home` |

### 使用方式

1. **分享到好友**：点击右上角 `···` → 转发 → 选择好友
2. **分享到朋友圈**：点击右上角 `···` → 分享到朋友圈

### 技术实现

每个页面的 `.js` 文件中添加了：
```javascript
onShareAppMessage() {
  return {
    title: '分享标题',
    path: '/pages/home/home',
    imageUrl: ''  // 留空使用页面截图
  };
}

onShareTimeline() {  // 朋友圈分享
  return {
    title: '分享标题',
    query: '',
    imageUrl: ''
  };
}
```

### 注意事项

1. ✅ **所有页面分享已启用**：右上角分享按钮现在应该是可点击状态
2. ⚠️ **tabBar 页面限制**：关卡地图页（`stage`）是 tabBar 页，分享路径不能带参数，统一跳转到首页
3. 📸 **分享图片**：目前使用默认截图，如需自定义封面图，可在 `imageUrl` 填入图片路径
4. 🎯 **分享路径可追踪**：游戏页/奖励页的分享带有具体关卡参数，好友点击可直达

### 如果还是不能分享

请检查：
1. **小程序后台设置**：登录 mp.weixin.qq.com → 设置 → 基本设置 → 确保"允许分享朋友圈"已开启
2. **体验版/正式版**：确保已提审并发布，开发版某些分享功能可能受限
3. **微信版本**：确保微信版本 ≥ 7.0.12（朋友圈分享需要）
