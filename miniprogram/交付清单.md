## 🚀 使用步骤

### 1️⃣ 解压文件
```
解压 → 得到 miniprogram/ 文件夹
```

### 2️⃣ 导入微信开发者工具
```
打开工具 → 导入 → 选择 miniprogram/ → 填 AppID → 完成
```

### 3️⃣ 配置域名白名单（真机必需）
```
登录 mp.weixin.qq.com
→ 开发 → 服务器域名 → downloadFile
→ 添加：
   https://images.unsplash.com
   https://www.pacdv.com
```

### 4️⃣ 编译运行
```
点击「编译」→ 在模拟器或真机预览
```

## ⚠️ 重要提示

### 必须配置的域名
```
downloadFile 合法域名：
  https://images.unsplash.com
  https://www.pacdv.com
```
**不配置的话真机无法加载图片和播放叫声！**

### AppID 填写位置
```
project.config.json 第 4 行：
"appid": "在这里填你的 AppID"
```

### 图标替换（可选）
```
assets/ 目录下的 6 个 PNG 文件
当前是占位图标，建议替换为正式设计
尺寸：81×81px
```
