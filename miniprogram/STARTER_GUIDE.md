### 游戏玩法

**认知类游戏（`recognize`）**：
1. 屏幕中央显示一个**大号 emoji**（如 🐶）
2. 下方显示 **4 个选项**（如：狗、猫、猪、兔子）
3. 点击正确的名字即可通关

### 年龄段顺序

现在主页从上到下的顺序是：
1. **🌱 启蒙班**（2-3 岁）— 新增
2. **🍎 学前班**（3-5 岁）
3. **🚂 幼儿园**（5-6 岁）
4. **🏫 小学一年级**（6-7 岁）

### 技术实现

**数据结构**（`utils/data.js`）：
```javascript
starter: {
  title: '启蒙班',
  emoji: '🌱',
  levels: [
    {
      type: 'recognize',     // 认知类型
      category: 'animal',    // 类别
      items: ['🐶狗', '🐱猫', '🐷猪']  // emoji+名字
    }
  ]
}
```

**游戏逻辑**（`pages/game/game.js`）：
- 新增 `buildRecognize()` 方法
- 从 `items` 随机选一个作为题目
- 拆分 emoji 和文字（正则 `/^(.+?)(.+)$/`）
- 生成 4 个选项（1 个正确 + 3 个干扰）
- 复用现有的 `onOptTap` 判断逻辑

**UI 设计**（`pages/game/game.wxml/wxss`）：
- 大号 emoji 居中显示（`font-size: 200rpx`）
- 带浮动动画（`animation: float`）
- 4 个选项按钮排列

### 扩展方式

如需添加更多认知内容，在 `data.js` 的 `starter.levels` 数组里追加：

```javascript
{
  name: '认蔬菜',
  emoji: '🥕',
  color: '#66BB6A',
  type: 'recognize',
  category: 'vegetable',
  items: ['🥕胡萝卜', '🥬白菜', '🍅西红柿', '🥔土豆']
}
```

### 注意事项

1. ✅ **已完全兼容现有功能**：不影响其他年龄段
2. ✅ **数据格式统一**：`emoji+名字` 组合（如 `🐶狗`）
3. ✅ **自动解锁**：默认启蒙班全开放，无需解锁条件
4. 📝 **可扩展**：支持任意类别（动物、水果、颜色、形状等）
